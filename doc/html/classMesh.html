Detailed Description

Defines a mesh.

A mesh is represented by a list of vertices and edges. A vertex is a set of 3 floats representing a (x,y,z) pair in R^3. An Edge is represented by a pair of indices into the vertex list. Vertices in edges are listed in clockwise order as they appear on the mesh. Once a mesh is built it is loaded onto the GPU so it can be rendered
